# Service Desk Analytics Dashboard (Excel)

## Overview
This project analyzes service desk ticket data using Excel to evaluate operational performance, track SLA compliance, and identify trends in ticket volume and categories. The goal is to transform raw ticket data into a clear, KPI-driven dashboard for decision-making.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar & Line)
- Excel Functions (COUNTIF, COUNTA, IF statements)

## Key Metrics Tracked
- **SLA Compliance Rate:** 34%
- Total Ticket Volume
- Open vs Closed Tickets
- Ticket Volume by Category
- Ticket Trends Over Time

## Key Insights
- SLA compliance is significantly below typical operational benchmarks, indicating delays in ticket resolution.
- Certain ticket categories contribute disproportionately to overall workload.
- Ticket volume shows variation over time, suggesting potential workload spikes.

## Dashboard Features
- KPI summary section for high-level performance tracking
- Category-based breakdown of ticket distribution
- Time-series visualization of ticket trends
- Clean, simplified layout for quick interpretation

## Purpose
Designed to simulate a real-world IT service desk reporting dashboard and demonstrate data cleaning, aggregation, and visualization skills using Excel.
